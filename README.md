## Features

- Static Site Generation ( SSG )
- Client Side Rendering ( CRS )
- Server Side Rendering ( SSR )
- Responsive Design
- Dark Mode
- SEO Optimization
- Pagination
- Infinitive Scrolling
- Material UI

## Technologies used

- Hook
  - built-in
    - useState
    - useEffect
    - useRouter
    - useContext
    - useReducer
  - cutsom
    - useDarkMode
- Style
  - FrameWork
    - TailwinCss
    - Material UI
    - Prettier Plugin TailwindCss
  - Icon
    - MUI Icon

## Run Locally

1. Clone the project

```bash
https://github.com/aliseyedi01/Movie-App.git
```

2. Go to the project directory

```bash
  cd Movie-App
```

3. Install dependencies

```bash
  npm install
```

4. Start the server

```bash
  npm run dev
```
